# Arcane Markdown

[English](./README.md) | 简体中文

一个为现代 Web 构建的强大的基于块的 Markdown 编辑器。

## 功能特性

- **基于块的编辑**：直观的拖拽界面用于组织内容。
- **丰富的内容支持**：
  - 标题、段落、引用
  - 列表（有序、无序、任务列表）支持智能分组
  - 代码块，支持语法高亮 (PrismJS) 和自动缩进
  - 数学公式 (LaTeX)
  - Mermaid 图表
  - 表格、折叠详情、图片
  - 提示框 (信息、警告、成功、错误)
- **实时预览**：所见即所得。
- **导出**：一键导出为标准 `.md` 文件。
- **多语言支持**：支持中文和英文界面切换。

## 快速开始

1. **安装依赖：**
   ```bash
   npm install
   ```

2. **启动开发服务器：**
   ```bash
   npm run dev
   ```

3. **打开应用：**
   在浏览器中访问 [http://localhost:3000](http://localhost:3000)。
